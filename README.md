# Kickstart Skills

Agent skills for creating tokens on [EasyA Kickstart](https://kickstart.easya.io), a fair launch launchpad on Base.

## Installation

```shell
npx add-skill easya-tech/kickstart-skills
```

Compatible with Cursor, Claude Code, Codex, and any agent that supports the [Agent Skills spec](https://agentskills.io).

## What's included

### kickstart

Create tokens on Kickstart using just `curl` and `cast` (Foundry). No SDK, no API keys, no npm required.

- Upload metadata via the hosted Kickstart API
- Deploy tokens with bonding curves on Base mainnet
- Zero dependencies beyond a wallet and Foundry
